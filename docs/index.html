Sadida — Report Portal

Help us improve the mod. Pick the flow that matches what you want to send.

🐛 Bug Report

A crash, a stack trace, or something visibly broken. Includes a field for your Godot log.

💡 Issue

Not a crash — confusing text, awkward UI, weird balance. Something that should be better.

🃏 Suggestion

Propose a new card, relic, potion, or power. Validated so we can tell what you mean.

Privacy note: submitter IPs are hashed (not stored raw) and rate-limited to 10 submissions per hour. We don't ask for any identifying information. The source for this portal is on GitHub.
